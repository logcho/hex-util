# hex-util

A lightweight C++ utility for converting between hex and various data types, including `string`, `int`, `uint8_t`, `uint16_t`, and `uint32_t` vectors.

## Features
- Convert strings and integers to hex and back
- Convert between hex and vectors of `uint8_t`, `uint16_t`, and `uint32_t`
- Simple and efficient implementation

## Installation & Compilation

### 🔹 Clone the repository
```sh
git clone https://github.com/logcho/hex-util.git
cd hex-utility
```

### Use wget
```sh
hex-util:
	wget https://github.com/logcho/hex-util/releases/download/v1.0.0/hex-util-v1.0.0.tar.gz
	tar xvf hex-util-v1.0.0.tar.gz && \
	rm hex-util-v1.0.0.tar.gz
